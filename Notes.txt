(Windows)
To check DB is running:
run services app and check postgres from list

(Must add postgres to path environment variable in windows):
can also run psql from cmd terminal to verify its running since psql talks to active DB

login to DB via psql and insert your password used via installation (Run from terminal)
command: psql -U postgres 

After logging in via psql command, run:
command: \list
